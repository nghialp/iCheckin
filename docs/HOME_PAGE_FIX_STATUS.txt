Problem 1: Schema Mismatch

  • Example: PLACE_FIELDS asked for 'category' but schema has no such field

Problem 2: Wrong Type Definitions

  • Should be defined on actual 'Place' type

Problem 3: Wrong Field Structure
  • CHECKIN_FIELDS assumed 'content' field on Checkin type
  • But actual structure: Checkin.comments[].content
  • Need to query nested comments to get content
