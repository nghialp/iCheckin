💡 ABOUT 'DISTANCE' FIELD
════════════════════════════════════════════════════════════════════════════

Why Not in Schema?
  • Distance depends on user's current location
  • Must be calculated client-side in real-time
  • User's coordinates: (userLat, userLng)
  • Place coordinates: (place.lat, place.lng)
  • Distance = Haversine(user, place)

How to Calculate:
  function calculateDistance(lat1, lng1, lat2, lng2) {
    const R = 6371; // Earth's radius in km
    const dLat = (lat2 - lat1) * Math.PI / 180;
    const dLng = (lng2 - lng1) * Math.PI / 180;
    const a = Math.sin(dLat/2) * Math.sin(dLat/2) +
              Math.cos(lat1 * Math.PI / 180) * 
              Math.cos(lat2 * Math.PI / 180) *
              Math.sin(dLng/2) * Math.sin(dLng/2);
    return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  }

Usage in Component:
  const [userLat, userLng] = getCurrentLocation();
  const placesWithDistance = places.map(place => ({
    ...place,
    distance: calculateDistance(userLat, userLng, 
                                place.lat, place.lng)
  }));
